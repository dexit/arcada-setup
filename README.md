# Arcada setup

> Make [**Arcada**](https://github.com/mehanix/arcada) the ***"floor plan creation software"*** run locally using `docker-compose`.

## Install / start

```
# Have Docker and Docker Compose installed
git clone git@github.com:perguth/arcada-setup.git
cd arcada-setup
npm start
```
